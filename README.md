##  Identificação Automática de Linguagem Falada no domínio imagem com Redes Neurais Convolutivas

[TOCM]

This repository contains the code for the dissertation "Identificação Automática de Linguagem Falada no domínio imagem com Redes Neurais Convolutivas", not currently published. 

The code is run through a jupyter notebook, and as of now (Sep 18th) there are 3 languages available: German, English and Spanish.
